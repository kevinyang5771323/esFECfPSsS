# 项目简介

## 核心功能
- 用户管理：用户注册、用户登录、个人信息管理、密码修改、角色权限控制
- 科室管理：科室信息录入、科室信息查询、科室信息修改、科室信息删除、科室状态管理
- 医生管理：医生信息录入、医生信息查询、医生信息修改、医生信息删除、医生排班管理
- 挂号管理：在线挂号预约、挂号记录查询、挂号取消、接诊状态管理
- 就诊管理：患者接诊、病历创建、病历查询、病历详情查看
- 处方管理：处方开具、处方查询、药品库存自动扣减
- 药品管理：药品信息录入、药品信息查询、药品信息修改、药品信息删除、库存预警管理、药品分类管理
- 统计分析：用户统计、医生统计、挂号趋势图表、科室挂号分布、数据可视化大屏

## 适用场景
- 医院日常运营管理
- 从挂号预约到就诊开方的全流程管理
- 数据可视化统计分析

## 技术特点
- 前后端分离架构
- 使用Java、Vue3、SpringBoot3等技术栈
- 支持多角色权限管理
- 提供数据可视化统计分析功能
